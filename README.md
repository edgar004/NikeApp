# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Desktop: 1024px
- Does not yet have support for mobile devices

## Colors

### Primary

- Black: #131313;
- Gray: #232323;
- Green: #9BDC28;
- Blue: #03A9F4;
- Pink: #E91E63;

## Typography

- Poppins
- Sans-serif

### Header and Titles

- Family: [Poppins](https://fonts.google.com/specimen/Poppins)
- Weights: 800

### Body and Paragraphs

- Family: [Poppins](https://fonts.google.com/specimen/Poppins)
- Weights: 400
